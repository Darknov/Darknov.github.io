about

Hi, I am junior web and game developer from Poland, currently living in UK. I am obsessed with creating new things; be it games, websites or apps. My obsession started in 2014 when I was still in High School. I had created some simple console games in C++ like Tic-Tac-Toe, Snake, Tetris or Conway's game of life. And oh boy it was hard in beginning. I also had started learning object oriented in Java, created simple rpg game from course in pure Java: Github Link

Then on certain polish site similar to reddit I found someone that was creating game in Javascript. I became fascinated how simple it was to create and quickly share something with others. I quickly created some dumb projects like simple clicker: Lumberjack Simulator

Or some projects that used Canvas: Space Arcade Game Which wasn't so easy to do without using frameworks. I had to create from scratch many things: "physics" and collisions for objects; animations; enemy's AI; map scrolling; correct refresh rate and what happens when it goes below 60fps; correct rotation of objects on canvas and so much more. And then I was 100% sure that it was something that I wanted to do in the future.

Now I am focusing on Web Development in HTML, CSS and Javascript. I am learning ReactJS + Redux and Web Design. I am also learning some back-end development in Node.js + Express.js.
